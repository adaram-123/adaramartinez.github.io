Caesar Cipher, AES-256-GCM, & Proxy

A Python-based tool that shows how TLS encryption hides application-layer content from packet sniffers like Wireshark, even when the application itself is sending a trivially simple Caesar cipher. The project has since grown to include an AES-256-GCM encrypted channel demo, an anonymizing proxy, and two levels of network threat detection.

Python Networking Security Cipher

Disaster Relief

A pathfinding algorithm that determines the optimal route through a set of regions to provide disaster relief to the maximum number of people at the lowest travel cost. Built using recursion and OOP principles in Java.
